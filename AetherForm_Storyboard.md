## 1. 브랜드 아이덴티티

**브랜드명**: AetherForm  
**브랜드 유형**: 가상 지속가능 패션 브랜드  
**브랜드 성격**: 미니멀 / 오가닉 / 바이오필릭 / 퓨처리즘 / 프리미엄

### 타겟
- 20~40대 감도 높은 소비자
- 지속가능성, 디자인, 소재의 질감을 중시하는 사용자
- 과한 상업성보다 절제된 미학과 브랜드 감성을 선호하는 층

### USP(차별점)
- 자연에서 영감 받은 **바이오필릭 무드**와
- 미래적이지만 차분한 **프리미엄 패션 미학**을 결합
- “친환경”을 직접적으로 강조하기보다, **소재와 분위기 자체로 지속가능성을 체감하게 하는 브랜드**

### 톤앤매너
- 깨끗한 화이트 / 베이지 / 소프트 내추럴 톤
- 여백이 많은 구성
- 느리고 부드러운 카메라 움직임
- 조용하고 고급스러운 분위기
- 포토리얼리즘 기반의 광고 스타일

## 5. 시행착오 및 프롬프트 수정 기록

### 1차 생성 결과
- 의류 형태가 불안정하게 표현됨
- 친환경 / 지속가능성 이미지가 충분히 드러나지 않음
- 전체 광고 비주얼의 완성도가 낮았음

### 문제점 분석
- 프롬프트가 시네마틱 분위기에는 집중되어 있었지만,  
  패션 광고에서 중요한 **소재감, 지속가능성, 브랜드 톤**이 약했음
- 결과적으로 장면이 너무 공상과학적으로 보였고,  
  AetherForm의 **프리미엄 eco-fashion 이미지**가 충분히 전달되지 않았음

### 피드백 요청 문구
> The image looks too sci-fi and lacks sustainability cues.  
> Please improve the prompt for a premium eco-fashion advertisement.

### 개선된 키워드
- premium sustainable fashion campaign
- biophilic design
- plant-derived materials
- luxury editorial photography
- subtle organic textures
- high-end commercial advertising

### 수정 후 의도
- 미래적 분위기는 유지하되,  
  **패션 광고로서의 현실감과 고급스러움**을 강화
- 친환경 요소를 직접적으로 설명하기보다  
  **소재, 질감, 분위기**로 자연스럽게 전달하도록 조정
- 브랜드 정체성이 더 명확하게 보이도록 키워드를 압축

### 개선 효과
- 브랜드 정체성 강화
- 광고 이미지 품질 향상
- 재생성 횟수 감소
- 크레딧 사용량 절감
- 최종 영상의 톤앤매너 일관성 개선

## 6. 씬별 스토리보드

### Scene 1. Biomimetic Space Introduction
- **장면 설명**: 밝은 흰색 구조물 안에 생체모방적 공간이 드러나며, 유기적 이끼와 식물성 섬유가 은은하게 빛남
- **목적**: 브랜드의 자연 친화적이고 미래적인 세계관 제시
- **화면 요소**: 화이트 아키텍처, soft beige accents, glowing moss, organic fiber details
- **카메라 움직임**: slow cinematic push-in
- **조명**: soft natural light
- **색감**: white / beige / soft green
- **AI 생성 요소**: 배경 비주얼, 질감, 공간 구성
- **오디오**: 잔잔한 앰비언트 BGM 시작

### Scene 2. Fashion Model Reveal
- **장면 설명**: 미니멀한 패션 모델이 등장하고, 식물성 섬유 기반의 의상이 강조됨
- **목적**: 제품의 소재감과 실루엣 전달
- **화면 요소**: refined plant-based fiber outfit, fabric texture, elegant drape
- **인물 연출**: 얼굴 중심이 아니라 의상 중심
- **모션**: 아주 미세한 고개 회전, 부드러운 바람에 따른 옷감 움직임
- **카메라 움직임**: 느린 접근 또는 고정된 고급 화보 느낌
- **AI 생성 요소**: 인물, 의상, 움직임
- **오디오**: 배경음 지속, 필요 시 아주 미세한 바람 소리

### Scene 3. End Card / Brand Reveal
- **장면 설명**: 흰색과 베이지 톤의 미니멀한 배경 위에 브랜드명과 슬로건이 정리되어 나타남
- **목적**: 브랜드 인지 강화 및 메시지 마무리
- **화면 요소**: subtle organic fiber forms, negative space, brand text overlay
- **브랜드 텍스트**:  
  - **AetherForm**  
  - **Sustainable Futurewear**
- **디자인 방향**: 고급스럽고 조용한 엔드카드
- **카메라/연출**: 정적인 마무리, 여백 중심
- **AI 생성 요소**: 배경과 텍스트가 어울리는 최종 비주얼
- **오디오**: 잔향이 남는 부드러운 종료
