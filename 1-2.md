# AetherForm 8초 AI 광고 초기 기획안

## 1. 초기 기획 개요

- **프로젝트명**: AetherForm 8초 AI 광고 제작
- **브랜드명**: AetherForm
- **광고 길이**: 약 8초
- **초기 제작 방식**: 씬별 이미지 생성 → 이미지 기반 영상화 → 편집 통합
- **예상 사용 도구**: ChatGPT / 이미지 생성 AI / 영상 변환 AI / CapCut
- **기획 방향**: 브랜드 세계관, 제품 이미지, 엔드카드를 각각 분리하여 제작

---

## 2. 초기 기획 의도

초기에는 AetherForm의 브랜드 메시지를 **씬 단위로 나누어 명확하게 전달하는 방식**으로 기획하였다.  
각 장면마다 역할을 분리함으로써 짧은 8초 안에서도 브랜드 정체성, 제품의 질감, 마무리 인상을 순차적으로 보여주는 것이 목표였다.

이 방식은 다음과 같은 장점이 있다고 판단했다.

- 장면별 의도를 명확하게 표현 가능
- 이미지 생성 단계에서 디테일 조정이 쉬움
- 씬별로 수정과 재생성이 가능함
- 최종 편집 시 구조를 유연하게 조합할 수 있음

## 8. 초기 기획의 장점

초기 씬별 제작 방식은 다음과 같은 장점이 있었다.

1. **장면별 통제가 쉬움**
   - 각 씬의 분위기와 구도를 따로 조정 가능

2. **수정이 용이함**
   - 특정 장면만 다시 생성하면 됨

3. **스토리텔링이 명확함**
   - 브랜드 → 제품 → 마무리 구조가 뚜렷함

4. **편집 자유도 확보**
   - CapCut에서 순서를 바꾸거나 자막을 조절하기 쉬움

## 9. 초기 기획의 한계

하지만 실제 제작을 고려하면서 다음과 같은 한계가 예상되었다.

- 씬별로 여러 번 생성해야 하므로 크레딧 소모가 큼
- 무료 크레딧을 제공하는 서비스로 온전히 영상을 만드는게 제한됨
- 이미지→영상 전환 과정에서 품질 손실 가능
- 편집 단계가 길어질 수 있음
- 짧은 8초 영상에서는 장면별 분할이 오히려 비효율적일 수 있음

이 때문에 이후에는 **단일 생성 방식**으로 제작 전략을 변경하게 되었다.

---

## 10. 초기 기획에서 최종 기획으로의 전환 이유

초기 기획은 구조적으로 명확했지만, 실제 제작 과정에서는 다음 이유로 변경이 필요했다.

- 무료 크레딧 문제
- 생성 효율성 문제
- 짧은 러닝타임에 적합한 한 번의 통합 생성 필요
- 브랜드 영상의 흐름을 더 자연스럽게 만들 필요
- 후반 편집 부담을 줄이고 싶었음

그래서 최종적으로는  
**씬별 제작 → 단일 생성 방식**으로 전환하였다.

---

## 11. 초기 기획 요약

초기에는 AetherForm 광고를 씬별로 분리하여 제작하는 방식으로 설계했다.  
이 방식은 장면마다 역할이 분명하고 수정이 쉬운 장점이 있었지만, 실제 제작 단계에서는 제공된 크레딧과 효율 문제로 인해 단일 생성 방식이 더 적합하다고 판단하였다.

즉, 초기 기획은 **“분리형 제작”**, 최종 기획은 **“통합형 제작”**으로 정리할 수 있다.
